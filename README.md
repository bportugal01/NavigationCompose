# NavigationCompose

![App Preview](https://github.com/bportugal01/NavigationCompose/blob/Master/image/app.gif)

Este é um projeto de exemplo que demonstra a navegação em um aplicativo Android usando o Jetpack Compose. O GIF acima mostra uma prévia do aplicativo em funcionamento.

## Visão geral

Este repositório contém um aplicativo de exemplo que utiliza o Jetpack Compose, uma biblioteca moderna de criação de interfaces de usuário para Android. O objetivo principal deste projeto é demonstrar como criar um sistema de navegação eficaz em um aplicativo Compose.

## Recursos

- Demonstração de navegação em um aplicativo Jetpack Compose.
- Uso de animações e transições suaves.
- Ilustração do fluxo de navegação em um aplicativo Compose.
